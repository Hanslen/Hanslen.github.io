For the new project, you do not need to do any setup. So for here, I will illustrate how to setup with the existing project.

  1. Download & unzip the SDK
  2. Add the SDKS to your app.

  3. Add the dependencies. Just click on targets -> Your app name -> and then the ‚ÄòBuild phases‚Äô tab. Expand ‚ÄòLink Binary with Libraries‚Äô as shown. The libraries which need to be added have shown on the parse website.
    Then connect your app to parse, do the same step like the website tutorial, I will not explain more about that. One thing still need to pay attention again is ‚Äúimport Parse‚Äù. If you come with an error called, no such module parse, the reason will be the path for the framework!
